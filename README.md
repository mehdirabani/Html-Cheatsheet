## ✨ نمای کلی

این پروژه یک **مرجع سریع و گرافیکی** از تگ‌های HTML است که به صورت جدول‌های دسته‌بندی شده طراحی شده. با استفاده از این ابزار می‌توانید:

- تگ‌های مهم HTML را سریع پیدا کنید
- کاربرد هر تگ را به زبان ساده فارسی ببینید
- نمونه کد و خروجی واقعی هر تگ را همزمان مشاهده کنید
- از طراحی چشم‌نواز و حرفه‌ای لذت ببرید
- در پروژه‌های خود یا تیم‌تان از آن به عنوان مستند سریع استفاده کنید

> **یادآوری:** این پروژه هنوز در حال توسعه است و همه تگ‌های HTML را پوشش نمی‌دهد. با کمک شما کامل‌تر خواهد شد.

## 🧭 راهنمای استفاده

### 📌 برای یادگیرندگان (چگونه از این جدول استفاده کنم؟)

1. **مشاهده تگ‌ها**  
   جدول به چند بخش تقسیم شده: تگ‌های پایه، سرعنوان‌ها، فرمت‌دهی متن، تگ‌های طراحی، رسانه، جدول و لیست. هر سطر شامل نام تگ، توضیح، نمونه کد و خروجی زنده است.

2. **جستجوی سریع**  
   از کلیدهای `Ctrl + F` (ویندوز) یا `Cmd + F` (مک) استفاده کنید و نام تگ یا توضیح مورد نظر را جستجو کنید.

3. **کپی کردن کد تگ**  
   روی هر `<code>` (متن آبی‌رنگ) کلیک کنید سپس `Ctrl + C` یا `Cmd + C` را بزنید. (در آینده قابلیت کپی خودکار با یک کلیک اضافه خواهد شد)

4. **مقایسه خروجی**  
   خروجی هر تگ در آخرین ستون جدول نشان داده شده است – برای تگ‌هایی مانند `<h1>`, `<mark>`, `<video>` بلافاصله نتیجه را می‌بینید.

5. **ذخیره به صورت PDF**  
   در مرورگر گزینه `Print` (Ctrl+P) را انتخاب کرده و به عنوان PDF ذخیره کنید. تمام استایل‌ها حفظ می‌شود.

### 🛠️ برای توسعه‌دهندگان (چگونه تگ جدید اضافه کنم؟)

برای افزودن تگ جدید به این مستندات، کافیست مراحل زیر را انجام دهید:

1. فایل `index.html` را در شاخه اصلی پروژه باز کنید.
2. بخش مربوط به دسته‌بندی مورد نظر (مثلاً `<h2>Text Formatting</h2>`) را پیدا کنید.
3. در جدول (`<tr>`) مربوط به آن دسته، داخل `<tbody>` یک سطر جدید با ساختار زیر اضافه کنید:

```html
<tr>
  <td><code class="code">&lt;YourTag&gt;</code></td>
  <td>توضیح فارسی کاربرد تگ</td>
  <td><pre><code>&lt;YourTag&gt;متن نمونه&lt;/YourTag&gt;</code></pre></td>
  <td><YourTag>متن نمونه</YourTag></td>
</tr>
```
4.فایل را ذخیره کرده و در مرورگر تست کنید.

5.تغییرات را commit و Pull Request بزنید (مراحل مشارکت در ادامه آمده).

## 🤝 مشارکت در پروژه
از تمام توسعه‌دهندگان فرانت‌اند و علاقه‌مندان به HTML دعوت می‌کنیم در بهبود این پروژه کمک کنند. راه‌های همکاری:

گزارش باگ: یک Issue جدید با برچسب bug باز کنید.

پیشنهاد تگ جدید: Issue با برچسب enhancement باز کنید و نام تگ + توضیح + نمونه کد و خروجی را بنویسید.

بهبود ظاهر: اگر ایده‌ای برای بهتر شدن طراحی (رنگ‌بندی، فونت، انیمیشن) دارید، Pull Request بزنید.

تکمیل تگ‌های缺失: از آنجا که پروژه هنوز همه تگ‌ها را پوشش نمی‌دهد، افزودن تگ‌های پرکاربرد دیگر بسیار ارزشمند است.

فرآیند مشارکت کد:

مخزن را Fork کنید.

یک Branch جدید بسازید:
git checkout -b feature/my-new-tag

تغییرات خود را اعمال کرده و Commit کنید:
git commit -m "feat: add <article> tag with example"

Push کنید:
git push origin feature/my-new-tag

یک Pull Request در گیت‌هاب باز کنید.

📌 تمام Pull Requestها باید به شاخه master ارسال شوند. حتماً پیشنهاد خود را به صورت واضح توضیح دهید.
