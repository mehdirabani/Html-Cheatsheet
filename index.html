Basic Tag

تگ توضیحات نمونه کد خروجی
<!DOCtype html> اعلام نسخه و فناوری html به مرورگر 
<!DOCtype html>
 غیرقابل مشاهده
<html> ریشه سند 
<html>

<head> اطلاعات فراداده 
<head>
 -----
<meta> اطلاعات اضافی مانند کارکتر ست 
<meta charset="UTD-8">
 -----
<title> نمایش و تنظیم عنوان صفحه 
<!DOCtype html>
<html>
  <head>
    <title> Html Cheatsheet </title>
  </head>
</html>
 نمایش عنوان انتخابی در tab مرورگر
<link> برای اتصال به css خارجی و موراد مشابه 
<!DOCtype html>
<html>
  <head>
    <link rel="stylecheet" href="style.css">
  </head>
</html>
 -----
<body> محتوای اصلی قابل مشاهده 
<!DOCtype html>
<html>
  <head>
    <link rel="stylecheet" href="style.css">
    <title> Html Cheatsheet </title>
  </head>
  <body>
    other Tag
  </body>
</html>
